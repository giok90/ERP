# Mini ERP System

This is a basic ERP system built with Java and MySQL.

## 📦 Technologies Used

- Java 17
- JDBC (MySQL Connector)
- MySQL database
- Maven (for build and dependencies)
- DAO pattern

## ✅ Features

- View all customers
- Search customer by ID
- Add new customer

## 🛠️ Setup

1. Make sure you have MySQL running locally.
2. Create a database named `erpdb`.
3. Update the credentials in `DBUtil.java` if needed.
4. Use Maven to run the project.

## 💻 Run the application

```bash
mvn exec:java
